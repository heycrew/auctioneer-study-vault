# 拍卖程序与流程

> 对应《拍卖法》第四章（第41-57条），掌握从委托到成交的全链条流程。

## 🎯 一句话概述

拍卖活动遵循严格的法定程序，每个环节都有明确的法律要求和操作规范，拍卖师必须全程合规操作。

## 🔄 全流程概览（待填充）

```
委托 → 公告与展示 → 竞买登记 → 拍卖实施 → 成交确认 → 结算交割
```

### 第一阶段：委托（第41-44条）
- 委托人提供身份证明和所有权证明
- 签订书面委托拍卖合同
- 合同必备条款（第42条）
- 委托人不得参与竞买（第22条）

### 第二阶段：公告与展示（第45-48条）
- 拍卖公告发布时间要求（拍卖日7日前）
- 公告内容要求（第46条）
- 标的展示时间要求（不少于2日）
- 展示期间的瑕疵告知义务

### 第三阶段：竞买登记（第49-51条）
- 竞买人资格审查
- 竞买保证金
- 竞买号牌发放

### 第四阶段：拍卖实施（第52-53条）
- 拍卖师主持
- 竞价规则
- 三声报价
- 落槌成交

### 第五阶段：成交确认（第54-55条）
- 签署成交确认书
- 成交确认书的效力

### 第六阶段：结算交割（第56-57条）
- 佣金计算与收取
- 拍卖笔录
- 标的交付和价款结算

## 📋 关键时限速记

| 事项 | 时限要求 | 法律依据 |
|------|----------|----------|
| 拍卖公告发布 | 拍卖日7日前 | 第45条 |
| 标的展示 | 不少于2日 | 第48条 |
| 成交确认书签署 | 拍卖成交后 | 第54条 |
| 拍卖笔录 | 拍卖结束后 | 第55条 |
| 档案保管期限 | 不少于5年 | 《拍卖管理办法》 |

## 🔴 2024年网拍新规专题（法〔2024〕238号）

> 最高人民法院2024年10月29日发布，12条新规直接影响司法拍卖流程。

### 拍卖前：法院调查义务强化
- 不得以"现状拍卖"为由免除调查职责（第1条）
- 严格审查租赁权、居住权真实性（第2条）——打击虚假长租约
- 核查建设工程价款优先受偿权（第3条）

### 拍卖中：公告与处置规则
- **公告禁止模糊表述**：不动产公告不得使用"占有不明"等（第6条）
- 限制无保留价"1元起拍"乱象（第5条）
- **被执行人自行处置**：二拍流拍后可申请自行处置（≤60日），支持"带封过户"（第7条）

### 拍卖后：交付与监管
- **严禁"不负责腾退"声明**（第8条）——这是刚性新要求
- 拍卖辅助机构六大"严禁"（第10条）
- 打击伪造文书、诱导高额佣金（第11条）

⚠️ 司法拍卖相关考题将以此新规为最新依据。
