# 拍卖类型与方式

> 掌握不同拍卖方式的规则、特点、适用场景和策略含义。

## 🎯 一句话概述

拍卖方式决定竞价规则，不同的拍卖方式产生不同的策略行为和结果，选择适当的拍卖方式是拍卖师的核心能力之一。

## 📋 四大基本拍卖类型（待填充）

### 1. 英式拍卖（增价拍卖）
| 属性 | 说明 |
|------|------|
| 竞价方向 | 由低到高 |
| 规则 | 竞买人依次出更高价格，最后无人加价时成交 |
| 适用 | 文物、艺术品、房地产 |
| 策略 | 竞买人存在"赢家诅咒"风险 |
| 中国常用名 | 增价拍卖 |

### 2. 荷兰式拍卖（减价拍卖）
| 属性 | 说明 |
|------|------|
| 竞价方向 | 由高到低 |
| 规则 | 拍卖师从高价开始降价，第一个应价者成交 |
| 适用 | 鲜花、鱼类等鲜活品 |
| 策略 | 竞买人需在"价格低"和"被别人抢先"之间权衡 |
| 中国常用名 | 减价拍卖 |

### 3. 密封投标拍卖（第一价格）
| 属性 | 说明 |
|------|------|
| 竞价方式 | 密封递交投标 |
| 规则 | 出价最高者按其出价支付 |
| 适用 | 政府采购、工程招标、土地使用权 |
| 策略 | 低于真实估值以避免"赢家诅咒" |

### 4. 维克里拍卖（第二价格密封投标）
| 属性 | 说明 |
|------|------|
| 竞价方式 | 密封递交投标 |
| 规则 | 出价最高者获胜，但只支付第二高价格 |
| 策略 | **诚实出价是最优策略**（理论上） |
| 适用 | 邮票、频谱牌照等 |

### 其他拍卖方式
- **电子拍卖**：通过网络平台进行
- **同步与异步拍卖**
- **组合拍卖**：多个物品打包拍卖
- **双向拍卖**：买卖双方同时报价
