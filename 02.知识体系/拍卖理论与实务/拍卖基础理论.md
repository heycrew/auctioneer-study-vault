# 拍卖基础理论

> 从经济学和法学双重视角理解拍卖的本质。

## 🎯 一句话概述

拍卖是通过公开竞价发现价格的市场机制，其核心功能在于解决信息不对称下的价格发现问题。

## 🔑 核心概念（待填充）

### 拍卖的经济学定义
拍卖是一种价格发现机制。当卖家不确定物品的市场价值（信息不对称）时，通过拍卖让买家之间的竞争来确定价格。

### 拍卖的法律特征
1. **公开竞价**：多个竞买人相互竞争
2. **最高应价**：价高者得（通常规则）
3. **特定物品或权利**：拍卖标的有特定范围
4. **买卖方式**：本质上是一种特殊的买卖合同

### 拍卖的功能
- **价格发现**：通过竞争确定市场公允价值
- **资源配置**：将物品配置给出价最高的人（最高效用）
- **交易透明**：公开竞价减少暗箱操作
- **价值实现**：帮助委托人实现物品的潜在最大价值

### 拍卖的历史沿革
- 古巴比伦：婚姻拍卖（约公元前500年）
- 古罗马：战利品拍卖（sub hasta，矛下拍卖）
- 17世纪荷兰：郁金香拍卖
- 18世纪英国：苏富比(1744)、佳士得(1766)
- 中国：改革开放后恢复拍卖业（1986广州成立第一家拍卖行）
