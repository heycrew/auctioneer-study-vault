## 🏁 阶段划分（倒推计划）

### 阶段一：基础搭建（第1-4周）
- [ ] 通读教材第一遍
- [ ] 建立完整知识 Wiki（12个核心页面全部填充）
- [ ] 每个页面完成第一轮主动回忆
- [ ] 开始每日间隔重复
- **里程碑**：能闭卷说出每个模块的知识框架

### 阶段二：深度理解（第5-8周）
- [ ] 所有核心概念完成费曼讲解
- [ ] 错题本中所有 ❌ 题清零
- [ ] 完成第一套模拟考试
- [ ] 薄弱环节识别并专项突破
- **里程碑**：能用自己的话解释90%的核心概念

### 阶段三：强化训练（第9-12周）
- [ ] 大量真题训练（至少10套）
- [ ] 模考得分率达到80%以上
- [ ] 主持实操模拟训练
- [ ] 案例分析专项训练
- **里程碑**：模考稳定达标，主持实操流畅

### 阶段四：冲刺（考前2周）
- [ ] 高频错题集中消灭
- [ ] 重点法条精准记忆
- [ ] 核心概念终极回顾
- [ ] 考前全真模拟
- **里程碑**：信心满满上考场
