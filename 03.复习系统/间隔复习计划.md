# 间隔复习计划

> 🧠 **Karpathy 核心原则**：间隔重复是长期记忆的唯一科学方法。
> 刚学的内容遗忘最快，必须在遗忘临界点之前回顾。
>
> 基于艾宾浩斯遗忘曲线，按 **1-2-4-7-15-30** 天间隔复习。

## 📐 间隔重复原理

```
遗忘曲线：
第1次学习 ──→ 20分钟后剩余58%
第1次复习 ──→ 1天后剩余33%
第2次复习 ──→ 4天后剩余20%
第3次复习 ──→ 7天后……
第4次复习 ──→ 15天后几乎永久记住
```

每次复习的时机应在"即将遗忘但尚未遗忘"的时刻。

## 📅 复习排期规则

| 轮次 | 距上次学习间隔 | 操作 |
|------|---------------|------|
| R0 | 当日 | 首次学习（摄入新知识） |
| R1 | 1天后 | 快速回顾 + 做主动回忆题 |
| R2 | 2天后 | 费曼讲解 + 错题回顾 |
| R3 | 4天后 | 穿插练习 + 真题模拟 |
| R4 | 7天后 | 综合复习 + 薄弱环节攻坚 |
| R5 | 15天后 | 模考检验 + 查漏补缺 |
| R6 | 30天后 | 终极回顾 + 考前冲刺 |
